# Update profile Picture

**URL** : `/api/user/password`

**METHOD** : `PATCH`

**Auth required** : YES

#### Headers:

```json
{
  "Auth": "<BEARER>__<TOKEN>"
}
```

#### Body:

```json
{
  "oldPassword": "...",
  "newPassword": "...",
  "confirmPassword": "..."
}
```
