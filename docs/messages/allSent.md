# All Sent Messages

**URL** : `/api/message/sent`

**METHOD** : `GET`

**Auth required** : YES

#### Headers:

```json
{
  "Auth": "<BEARER>__<TOKEN>"
}
```

#### Query params:

```json
{
  "limit": <NUMBER>,
  "page": <NUMBER>,
  "sort":// <1> or <-1>  1 => ASC  -1 => DESC
}
```
