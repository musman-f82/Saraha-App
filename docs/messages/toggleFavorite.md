# Toggle Favorite Status

if message favorite status = `true` becomes `false` and vice versa

**URL** : `api/message/:messageId/favorite`

**METHOD** : `PATCH`

**Auth required** : YES

#### Headers:

```json
{
  "Auth": "<BEARER>__<TOKEN>"
}
```

**code** : **`200`**
