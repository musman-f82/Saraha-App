# Signup

**URL** : `/api/auth/signup`

**METHOD** : `POST`

#### Body:

```json
{
  "userName": "yhya101",
  "email": "yhya@email.com",
  "password": "123789@Yy",
  "confirmPassword": "123789@Yy",
  "gender": "male"
}
```

**code** : **`201`**
